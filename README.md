# poto
boto3 wrapper for DL training and inference

# install
`pip install git+https://github.com/POZAlabs/poto`

## config must contains
```
{
    "service_name" : "s3",
    "endpoint_url" : "https://kr.objectstorage.ncloud.com",
    "region_name" : "kr-standard",
    "access_key" : "...",
    "secret_key" : "..."
}
```
